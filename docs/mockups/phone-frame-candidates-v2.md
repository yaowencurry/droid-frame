# 手机壳候选风格 V2

这版按用户反馈重做：不要粗边框，不要假厚壳，参考小米/OPPO 系统“带壳/套壳截图”的精致窄边效果。

## 参考方向

- OPPO / ColorOS：截图后进入编辑页可选“套壳”，系统自动识别机型并渲染手机外壳。重点是 3D 手机外壳和分享质感。
- 小米 / MIUI：系统曾提供“带壳截图”，截图编辑页里可一键添加当前机型外观。
- Snapmod / Device Frame Generator：Android 生态常见的带壳工具，机型库丰富，但部分旧素材边框偏厚。
- 在线样机工具：适合参考构图、背景色、阴影和导出比例，但素材授权需谨慎。
