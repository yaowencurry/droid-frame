# 手机壳候选风格 V3

这版继续按用户反馈收窄边框：目标不是“画一个手机壳”，而是接近用户图二的高级导出图。

## 筛选结论

- 小米 MIUI 曾全量推送“带壳截图”，入口是截图后点击“带壳”，可一键嵌套对应机型外壳。
- OPPO / ColorOS 的“套壳”入口在截图后的编辑页，效果更偏 3D 外壳和分享图质感。
- 直接拿网络手机壳素材存在授权风险，App 内先用程序化复刻的细金属边框，再按用户确认做风格收敛。

## 推荐

先让用户确认 A / B / D 的方向。落地时默认使用 A，导出比例默认 3:4，背景默认 `#F3F5EA`。
